
# OasisBio .github

This repository contains the GitHub community health files for the Oasis-Company/OasisBio project.

## About OasisBio

**OasisBio** is an **open identity context infrastructure for the AI era**.

The AI era needs a new identity layer. Every time you use a new AI assistant, you re-explain who you are. OasisBio fixes this — create your structured identity once, let machines understand you everywhere. Build structured, machine-readable identity profiles that AI assistants and applications can instantly understand. Stop reintroducing yourself to every AI — create your context once, use it everywhere.

> "A Trans-Era Identity Agreement"

## What's Included

This `.github` repository hosts:

- **Issue Templates**: Standardized issue templates for bug reports, feature requests, and other contributions
- **Pull Request Templates**: Guided templates for creating pull requests
- **Workflows**: GitHub Actions workflows for CI/CD and automation
- **Community Guidelines**: Code of conduct, contributing guidelines, and more

## Tech Stack

- **Frontend**: Next.js, TypeScript, Tailwind CSS
- **Backend**: Next.js API Routes, Supabase
- **Database**: Prisma ORM
- **Deployment**: Vercel
- **Storage**: Cloudflare R2 (for 3D models)

## Quick Start (Main Repository)

To run the main OasisBio project:

```bash
# Prerequisites: Node.js 18+, Supabase project, Cloudflare account with R2
git clone https://github.com/Oasis-Company/OasisBio.git
cd OasisBio
npm install
npm run dev
# Open http://localhost:3000
```
